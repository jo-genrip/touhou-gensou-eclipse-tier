# 「東方幻想エクリプス」想起カード　tier表作成ツール
想起カードと呼ばれるゲーム内アイテムの評価を簡単にできるtier表を画像をドラッグ＆ドロップで作成し、画像を保存できるツールです。
私も共同で制作しているゲームのwikiへの埋め込み用に作りました。
## 公開URL(実際にwikiの方で使用可能)
[[httpsw.atwiki.jpgenlippages572.html](https://w.atwiki.jp/genlip/pages/572.html)]

◦実際にwikiの方で稼働しているtier表メーカーです。ちゃんと作成から保存までできます。

## 主な機能
- 自動で画像を取得：同wiki内にある元となるtier表で選択するための想起カードを取得します。
- ドラッグ操作可能：スマホでもPCでも操作可能です。
- 行のカスタマイズ：既存にあるランクだけでなく、新しい行の追加・色替えなどができます。
- 画像保存：画像保存のボタンを押すだけで、行の右側の様々な機能のボタンを一度隠して横6枚並びの画像を生成して保存します。
## ファイル構成
開発段階では1つのファイルにすべてまとめていましたが、見やすさと管理のしやすさから機能を3つのファイルに分割して構成しています。

- `index.html`  ツールの骨組み・レイアウト
- `style.css`  PC・スマホ・画像書き出し用のデザイン設定
- `script.js`  画像のスクレイピング、並び替え（SortableJS）、画像化（html2canvas）のロジック
## 工夫した点
- 操作性：直感的な操作ができるようにドラッグ&ドロップで操作できるようにしました。
- 画像保存：作成してスクリーンショットをユーザ側にさせるのではなく、ブラウザ上でスマホ版でもPCでも同じ画像サイズでの保存をできるようにした。
- ランクのバーごと移動；tier表を作成する中で、画像のあるパレットから一番上のランクへ配置するということがしづらくなってしまうことがありました。
ですがランク内の画像ごと移動できるようにしたことで一旦最下層に置いてから後で上に移動することができ、その点の使いづらさを改修することができました
- 画像のあるパレットの移動方法；下にある画像が余白なしに入っているとスクロールが困難になってしまうので、スクロールバーの設置と余白を入れてスクロールを容易にしました。
